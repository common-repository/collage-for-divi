== Description ==

<strong>Divi Collage - The Easiest Collage Maker For Divi.</strong>

Divi Collage is a Divi module which lets you create beautiful collages easily. The module comes with a bunch of ready-made layouts. Just pick a layout and fill your images.

Visit <a href="http://demo.cakewp.com/divi-collage/?utm_source=wp-repo&utm_medium=link&utm_campaign=readme" target="_blank" rel="">Divi Collage</a> for more info.

<strong>Important Note:</strong> Divi is built on-top of Divi Theme's API. This plugin requires Divi Theme or Divi Builder plugin to function. It is a Divi module.
